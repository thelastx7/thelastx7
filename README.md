## 🧠 About Me

I'm a **Cybersecurity Engineering student** based in Morocco, focused on Offensive Security, Penetration Testing, Exploit Development, Reverse Engineering, Malware Analysis, AI Security, and Secure Software Development.

- 🔍 I enjoy understanding how systems work internally, discovering vulnerabilities, writing security tools, and automating repetitive tasks.
- 🛠️ I prefer building practical projects instead of only studying theory.
- 🎯 **Long-term goal:** becoming an elite Offensive Security Engineer capable of advanced penetration testing, exploit research, reverse engineering, and AI-assisted cybersecurity.
- 🌱 Also interested in: Artificial Intelligence, Automation, Linux, Cloud Security, Networking, Web Security, Active Directory, Red Teaming, Digital Forensics, Threat Hunting, Secure Coding, Bug Bounty, and CTFs.
- 💡 **I value:** continuous learning, clean code, curiosity, discipline, technical excellence, and open-source contribution.

> *"I enjoy breaking systems to understand how to build them better."*
